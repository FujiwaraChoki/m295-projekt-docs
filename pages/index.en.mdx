# M295 ToDo Project

## Introduction

This is the final project for module 295 that was done at ZLI.
The project consists of a backend for a ToDo app that was programmed using the Express.js
Framework was programmed.

## Requirements

- ✅ README.md
- ✅ [Documentation](https://m295.samihindi.com)

## Technologies

- **Language**: JavaScript
- **Runtime**: Node.js
- **Frameworks**: Express.js, Express-Sessions

## Installation

1. clone repository
```bash copy
git clone https://github.com/FujiwaraChoki/m295-projekt.git
```
2. install dependencies
```bash copy
cd m295-project
install npm
```
3. start server
```bash copy
start npm
```
