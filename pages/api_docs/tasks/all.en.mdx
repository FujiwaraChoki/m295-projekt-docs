# /

When the client sends a request to this endpoint, the server returns a JSON Array
containing all the tasks.

## Parameters

***None***

## Responses

| Status | Meaning | Description |
| ------ | ------- | ----------- |
| **200** | OK | The tasks were successfully returned |
| **403** | Unauthorized | The User isn't logged in |
| **500** | Internal Server Error | Something went wrong on the server |

## Example

### Request (JavaScript)

```javascript copy
const url = 'http://localhost:3000/tasks';

fetch(url)
    .then(res => res.json())
    .then(json => console.log(json))
    .catch(err => console.error(err));
```
